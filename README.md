A simple website that provides a list of plugins for [Revenge](https://github.com/revenge-mod/revenge-bundle?tab=readme-ov-file#-revenge), indicating their current working status. 

Link: https://plugins-list.pages.dev/

## Building

Instructions to set up and run the project locally.

### Prerequisites

-   [Bun](https://bun.sh/) must be installed.

### Installation & Development

1.  Clone the repository:
    ```bash
    git clone https://github.com/Purple-EyeZ/Plugins-List.git
    cd Plugins-List
    ```
2.  Install dependencies:
    ```bash
    bun install
    ```
3.  Run the development server:
    ```bash
    bun run dev
    ```
    The site will be available at `http://localhost:3000`.
