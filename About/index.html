Plugin Status

The status of each plugin is based on whether it works on the latest stable version of Discord.

  • Working: The plugin functions as expected with no known issues.
  • Warning: Some features of the plugin may not work properly. It might be partially broken or have minor bugs.

  • Broken: The plugin doesn't function correctly or causes major issues.

Unproxied Plugins

All the plugins listed here are unproxied, meaning their developers can update or modify them at any time. This also means they haven't been reviewed or verified. If you don't trust these plugins, don't install them.

Feedback and Suggestions

If you see a plugin with an incorrect status or want to submit a new one (or something else), please open an issue on the GitHub repository.
